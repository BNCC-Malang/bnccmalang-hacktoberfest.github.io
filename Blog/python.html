Python

Apa itu Python?

Python adalah bahasa pemrograman yang dapat mengeksekusi serangkaian pernyataan tujuan umum secara langsung (interpretatif) menggunakan metode berorientasi objek. Python adalah bahasa pemrograman yang paling mudah dipahami. Python dibuat oleh seorang programmer Belanda bernama Guido Van Rossum.

Sejarah Perkembangan Python

Python dibuat dan dikembangkan oleh Guido Van Rossum, seorang programmer dari Belanda. Produksi berlangsung pada tahun 1990 di kota Amsterdam, Belanda. Pada tahun 1995, Guido Van Rossum mengembangkan lebih lanjut Python agar kompatibel. Ada juga pembaruan versi Python di awal tahun 2000 hingga mencapai versi 3 hingga hari ini. Pemilihan nama Python sendiri berasal dari acara TV ternama bernama Mothy Python Flying Circus, acara sirkus favorit Guido van Rossum.

Kelebihan dan Kekurangan Python

Banyak orang yang tertarik dengan Python karena dianggap mudah dipelajari bahkan untuk pemula. Kode yang terdapat dalam mudah dibaca dan, karena banyaknya pustaka standar, kode tersebut dapat menjalankan banyak fungsi kompleks tanpa masalah. Pengembangan program dengan Python juga dapat dilakukan dengan cepat dan membutuhkan lebih sedikit kode. Bahkan Python dapat menyederhanakan program penskalaan yang sangat kompleks. Python sendiri adalah cross-platform dan multi-sistem yang kompatibel dan memiliki sistem manajemen memori otomatis seperti Java. Sayangnya, Python berjalan cukup lambat. tidak memiliki dukungan untuk pengembangan platform Android dan iOS. Python juga memiliki batasan akses database. Selain itu, Python tidak cocok untuk tugas yang membutuhkan banyak memori dan pekerjaan multicore/multiprosesor. Ini adalah ikhtisar singkat tentang bahasa pemrograman Python yang dapat Anda rujuk. Bagi para developer atau pemrograman pemula, sangat disarankan untuk mempelajari bahasa pemrograman karena dianggap mudah. Python juga dapat digunakan dalam pengembangan perangkat lunak, aplikasi smartphone, program GUI, program CLI, Internet of Things, game, dan lain-lain.
